    Denotes a State Transition with the Postgresql State Machine that should be impossible.

    The Postgresql 3.0 Protocol describes serveral specific connection State Machines depending on the phase of the connection - StartupPhase ( made up of Authentication Phase and Server Process Startup Phase ), SimpleQuery Phase. During these phases, it is possible for a Postgresql Server to transmit a Message that does not make sense given the current State of the Connection. In practice, these should be extremely rare.

    http://www.postgresql.org/docs/current/static/protocol-flow.html
